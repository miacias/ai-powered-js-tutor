# AI-Powered JavaScript Tutor

[![License](https://img.shields.io/badge/license-MIT-blue?logo=github)](https://github.com/miacias/ai-powered-js-tutor/blob/main/LICENSE)

## Description

This app is a JavaScript Consultant application where users could ask any question related to JavaScript concepts. The app will function for as long as the API key remains valid!

## Usage

[Deployed site here!](https://ai-powered-js-tutor.vercel.app/)

First, navigate to the site, then enter your question into the input box. Finally, hit "Ask!"

<img src="https://github.com/miacias/ai-powered-js-tutor/assets/113298622/59a8902c-a86c-429e-9720-d400ed1eb6cb" alt="Homepage for your JavaScript Tutor with a blank form to ask a question" width="80%"/>

The tutor will respond below in approximately 250 characters of combined code and a brief explanation.

<img src="https://github.com/miacias/ai-powered-js-tutor/assets/113298622/3e588dff-a970-4df3-9f18-6a9bf54c418e" alt="Homepage for your JavaScript Tutor with a question in the form and a response below" width="80%"/>
